Food Delivery Backend in Java
Using various concepts of OOP and Exception Handling

How to Run:

1. First add a restaurant through the Admin portal:

  ~Username: admin

  ~Password: 1234

  ~Add a restaurant (enter restaurant name and password)

  ~View restaurants

  ~View customers

2.Add food items through the Restaurant portal:

  ~The restaurant username is the same as the restaurant name

  ~The password is provided by the admin while adding the restaurant

  ~Add food items with their respective prices

  ~Add a discount coupon (enter the coupon code if you want to add one)

  ~Select the respective option to view orders

  ~View total revenue

3.To place an order:

  Enter your name

  ~Choose the restaurant of your choice

  ~Select food items from that restaurant

  ~If you have a coupon code, apply it

  ~Confirm your order
